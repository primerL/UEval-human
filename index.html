📝 How to Complete Evaluation

  1. Click on your assigned evaluation set above
  2. Compare the reference content (left) with AI-generated content (right)
  3. Rate all 6 criteria on a 0-5 scale and give concise reasoning
  4. Click “Save Evaluation” for each sample (progress auto-saves)
  5. Use arrow keys (← →) or navigation buttons to move between samples
  6. When finished, click “Download All Evaluations” to export your JSON file
  7. Send the downloaded JSON back to the coordinator
